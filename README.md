# CopySignature
copy signature from signed pe to no signed pe

Usage：
    1. Select Nosigned pe，like exe、dll、sys
    2. Select Singed PE，like exe、dll or sys
    3. Clicked Copy！

result：
    Create File named Nosigned Name.copysig


Support:
    Copy 32bit to 32bit
    Copy 64bit to 64bit
    You can also copy 64bit PE to 32bit PE
